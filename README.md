# FakeChat

A fake chat generator, styled after **Telegram** and **Viber** (plus an Instagram - not finished), running entirely in the browser. Great for mockups, memes, presentations, or UI demos - not meant to pass off a conversation as real.

## Features

**Chat**
- Send messages, styled like the original app's bubbles
- Reply to a message (with a preview above the input, Telegram only)
- Context menu on messages: Reply, Delete, Edit
- Delete all messages, with a confirmation popup
- Switch roles (who's "me" and who's the other person) with one click
